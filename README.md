# BOS Votes Snapshots

## Installation

```bash
git clone https://github.com/EOS-Nation/bosvote-snapshots.git
cd bosvote-snapshots
npm install
```

## Quick Start

```bash
npm start
```

## ENV Variables

```env
BLOCK_INTERVAL=50
CHAIN="bos"
NODEOS_ENDPOINT="http://localhost:8888"
ELASTICSEARCH_ENDPOINT="http://localhost:9200"
```
