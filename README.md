# BOS Votes Snapshots

## Installation

```bash
git clone https://github.com/EOS-Nation/bosvote-snapshots.git
cd bosvote-snapshots
npm install
```

## Quick Start

```bash
npm start
```

## ENV Variables

```env
CHAIN="eos"
NODEOS_ENDPOINT="http://localhost:8888"
CONTRACT_FORUM="eosio.forum"
DELAY_MS=10
```
